# Desafio DIO: Conta Bancária

Este projeto foi desenvolvido como parte de um desafio educacional proposto pela plataforma Digital Innovation One (DIO). O objetivo do projeto é criar um programa simples em Java que simula a criação de uma conta bancária, solicitando informações do usuário e exibindo os dados cadastrados.

## Funcionalidades

- Solicitar o nome do cliente.
- Solicitar o número da conta.
- Solicitar o número da agência.
- Solicitar o saldo inicial.
- Exibir uma mensagem de boas-vindas com os dados fornecidos.

## Tecnologias Utilizadas

- Linguagem de Programação: Java
- Ferramenta de Entrada: `Scanner` para leitura de dados do usuário.

## Como Executar o Projeto

1. Certifique-se de ter o Java instalado em sua máquina.
2. Compile o arquivo `ContaTerminal.java` localizado na pasta `src`.
3. Execute o programa e siga as instruções exibidas no terminal.

```bash
# Para compilar:
javac src/ContaTerminal.java -d bin

# Para executar:
java -cp bin ContaTerminal
```
